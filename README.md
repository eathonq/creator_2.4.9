# 项目目录结构说明
```
    # assets
        # core              # 基础包
            # common            # 公共脚本
            # i18n              # 语言包脚本
            # platform          # 多平台相关脚本
                # wechat            # 微信平台脚本
            # ui                # UI脚本
        # game              # 游戏实现
            # models            # 模型
            # prefabs           # 预制体
            # scripts           # 脚本
        # resources         # 资源
            # fonts             # 字体
            # i18n              # 语言包
            # images            # 图片
            # sounds            # 音效
            # transitions       # 过渡动画(js版本，部分不兼容)
```
